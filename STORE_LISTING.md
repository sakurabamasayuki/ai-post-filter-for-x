## 📌 拡張機能名

```
AI Post Filter for X
```

## 📌 概要文 (132文字以内)

```
X(Twitter)のタイムラインからAI生成投稿を検出。3段階の判定エンジンでAI生成・インプレ稼ぎを自動でぼかし・非表示にします。
```

## 📌 説明文 (詳細)

```
X(旧Twitter)のタイムラインに溢れるAI生成投稿やインプレ稼ぎ投稿を自動で検出し、見やすく整理する拡張機能です。

【🎯 主な機能】

✅ AI生成投稿の自動検出
ヒューリスティック判定、アカウント信号分析、Claude AIによる3段階の検出エンジンで高精度に判定します。

✅ 3種類の表示モード
・ぼかし表示: AI判定投稿にブラー効果を適用
・完全非表示: AI判定投稿をタイムラインから除外
・ラベルのみ: スコアバッジで判定結果を表示

✅ 判定詳細の可視化
バッジをクリックすると、AI/インプレ稼ぎ/人間 の3カテゴリスコア、検出されたパターン、判定理由を視覚的に確認できます。

✅ ホワイトリスト/ブラックリスト
特定のアカウントを常に表示・常に非表示に設定可能。ワンクリックで管理できます。

✅ カスタマイズ可能なしきい値
50%〜100%でAI判定の感度を調整できます。

✅ ダーク/ライトモード対応
お好みのテーマでご利用いただけます。


【🛡️ プライバシー】

・投稿内容は判定のためにのみ一時的に使用され、保存されません
・個人情報は一切収集しません
・全ての通信はHTTPS暗号化されています

詳細: https://github.com/sakurabamasayuki/ai-post-filter-for-x/blob/main/PRIVACY.md


【⚡ Beta版について】

現在Beta版として公開しています。フィードバックや機能要望をお待ちしています!

・無料: 1日あたり十分な検出回数を利用可能
・Pro版: 近日公開予定(無制限利用、優先サポート)


【💬 サポート】

バグ報告・機能要望はGitHubまで:
https://github.com/sakurabamasayuki/ai-post-filter-for-x


【🔧 動作環境】

・X(twitter.com / x.com)のみで動作
```

## 📌 権限の正当性 (Justification)

### `storage` 権限
ユーザー設定(表示モード、しきい値、ホワイトリスト等)を保存するために使用します。

### `host_permissions: x.com, twitter.com`
タイムライン上の投稿テキストを取得してAI判定するために必要です。X以外のサイトではアクセスしません。

### `host_permissions: workers.dev`
Cloudflare Workersで実装した判定APIへのアクセスのために必要です。

## 📌 提出フロー

1. https://chrome.google.com/webstore/devconsole にログイン
2. 「新しいアイテム」をクリック
3. `ai-post-filter-beta-v0.1.0.zip` をアップロード
4. 上記の情報を入力
5. スクリーンショット3枚をアップロード
6. プライバシー設定にて「データ収集」を申告
7. 「審査用に送信」

審査期間: 通常1〜3日
